# Camera Labs

Repository con progetti GNS3 per esercitazioni di networking.

## Setup

### Scarica la VM
Sono disponibili due macchine virtuali preconfigurate (formato `.ova`):
- **ARM**: per Mac con processore Apple Silicon
- **x86**: per sistemi x86-64

Scarica dalla cartella Drive:
https://drive.google.com/drive/folders/1goXPwuiKH1CA33rT-8koiz5hZeerZN5b?usp=drive_link

### Credenziali per la VM
```
Utente: user
Password: user
```

### Primo avvio
1. Importa la VM nel tuo hypervisor (VirtualBox, Parallels, ecc.)
2. Avvia la VM
3. Sulla scrivania troverai l'icona di **GNS3**

## Progetti

I progetti pronti sono nella cartella `gns3_projects/`. Importali direttamente in GNS3 tramite il menu *File > Import portable project*.

I file hanno estensione `.gns3project` e sono pronti all'uso, senza bisogno di modifiche.
